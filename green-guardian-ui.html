01 — App Landing, Login & Account Creation

The first screens a user
will ever see

The entry flow sets the tone for the entire app experience. The landing page leads with Green Guardian's brand identity, followed by a clean login and account creation flow that removes friction and focuses the user on one action at a time.

The above prototype is for the App Landing page, Login page and Create an Account page.

02 — Onboarding UI

Showing users what's possible
before they explore

The onboarding sequence introduces users to Green Guardian's special and core features — giving new users a clear mental model of what they can do before they dive in. Each screen highlights one distinct feature with a strong visual and minimal, action-oriented copy.

The above Onboarding UI highlights the special and main features that the app has to offer.
